# Open Source Contribution Ledger

Log of contributions to public repositories, libraries, and tools.

## Purpose
To detail participation in global open-source projects, demonstrating code review feedback loop execution, test parity validation, and technical collaboration.

## Usage Conventions
- Record all contributions using `templates/career/open-source-contribution.md`.

## Contribution Guidelines for Future Self
1. Prioritize code quality and formatting over contribution volume.
2. Read the project's contributing rules (`CONTRIBUTING.md`) before writing code.
3. Be respectful and implement feedback from reviewers promptly.
