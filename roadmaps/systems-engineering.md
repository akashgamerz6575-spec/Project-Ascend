# Systems Engineering Roadmap

Outline of technical milestones to transition from application developer to system systems engineer.

## Milestones
- [x] Phase 1: High-level language proficiency (JavaScript/Python)
- [/] Phase 2: Lower-level language fundamentals (C++)
  - [x] Memory layout and variable declarations
  - [x] Value copy vs. assignment semantics
  - [x] Reference aliases and binding constraints
  - [ ] Pointer dereferencing and address manipulation
  - [/] Function stack frames and recursive layouts
- [ ] Phase 3: Relational databases deeper dive (PostgreSQL internals, indexes)
- [ ] Phase 4: Systems internals (Virtual Memory, Process management)
- [ ] Phase 5: Containerization (Docker, Kubernetes)
