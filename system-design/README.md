# System Design Case Studies

Case studies in High-Level Design (HLD) and Low-Level Design (LLD).

## Purpose
To detail the architectures of large-scale distributed systems and structural design pattern implementations.

## Usage Conventions
- Classify case studies under:
  - `system-design/hld/` for distributed networks, load balancers, caching, database replication.
  - `system-design/lld/` for class layouts, interfaces, design patterns, and SOLID implementations.

## Contribution Guidelines for Future Self
1. Avoid generic diagrams. Specify database engines, data formatting, and caching policies used.
2. In LLD design patterns, provide concrete, compile-ready code snippets.
