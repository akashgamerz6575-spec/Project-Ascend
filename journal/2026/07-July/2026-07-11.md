## Focus Areas & Learnings
- **Technical Concepts Explored:** Function execution lifecycles, memory scopes, parameters passing (value vs reference), Call Stack structure (LIFO), stack frames layout, and stack overflow causes.
- **Key Takeaways:**
  - Local variables reside inside stack frames, meaning their lifetime is strictly tied to function execution.
  - Recursion utilizes nested function calls, consuming stack memory linear to the recursion depth.
  - A reference parameter acts as a direct alias to the caller's memory address, avoiding copy overhead.
