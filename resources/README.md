# Engineering Resources & Reference Materials

A central location for standardized checklist collections, cheatsheets, and reference materials.

## Purpose
To compile and maintain high-quality reference sheets (Git commands, Docker setups, SQL queries) that optimize execution speeds and reduce lookup times.

## Usage Conventions
- Add reference sheets as markdown files inside `resources/`.
- Ensure cheat sheets are organized logically by tech stack or concept.

## Contribution Guidelines for Future Self
1. Avoid copy-pasting massive internet cheat sheets. Only record commands and steps that have been locally tested and verified.
2. Keep configuration snippets up to date with the runtimes specified in `docs/system-setup.md`.
