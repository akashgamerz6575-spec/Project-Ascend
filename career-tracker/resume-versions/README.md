# Resume Versions

A directory of chronological resume iterations and tailoring strategies.

## Purpose
To maintain structured copies of resumes tailored to different job profiles (e.g., backend systems, applied AI developer, full-stack engineer), tracking the changes and application performance.

## Usage Conventions
Copy the template from `templates/career/resume-version.md` to log changes when updating resumes.
