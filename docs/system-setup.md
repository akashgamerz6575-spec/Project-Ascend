# System Setup Guide

This document maintains environment configurations, runtime versions, and steps to bootstrap the development system.

## Primary Environment Details
- **OS:** Ubuntu 22.04 LTS (WSL2)
- **Node.js:** v18.16.0
- **PostgreSQL:** v15.3
- **C++ Compiler:** GCC v11.3.0

## Bootstrapping Steps
1. Clone the repository.
2. Run database migrations.
3. Configure the local `.env` environment variables.
