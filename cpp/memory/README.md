# Memory Management in C++

Stack vs Heap layouts, manual allocations, custom allocators, and hardware memory hierarchies.

## Core Concepts
- **Stack Memory:** Automatic duration, fast allocations, frame boundaries.
- **Heap Memory:** Dynamic duration, manual allocations via `new`/`delete` or `malloc`/`free`.
- **Memory Safety:** Double free errors, memory leaks, dangling references, buffer overflows.
