# Object-Oriented Programming (OOP) in C++

Class structures, access modifiers, constructors, destructors, inheritance hierarchies, and dynamic polymorphism.

## Core Concepts
- **Encapsulation:** Access control specifications (`public`, `protected`, `private`).
- **Inheritance:** Single, multiple, and virtual base classes to prevent diamond inheritance.
- **Polymorphism:** Method overriding, static/dynamic binding, and virtual tables (vtable).
