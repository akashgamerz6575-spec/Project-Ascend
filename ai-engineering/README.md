# Artificial Intelligence & ML Engineering

Notebooks, data pipelines, training schedules, and inference engines.

## Purpose
To document applied AI and ML engineering implementations, including modeling data preprocessing pipelines, deep learning architectures, and Large Language Model agent configurations.

## Usage Conventions
- Group items by phase:
  - `ai-engineering/classical-ml/` for regression, classification, clustering.
  - `ai-engineering/deep-learning/` for CNNs, RNNs, Transformers.
  - `ai-engineering/llm-agents/` for RAG pipelines, API orchestration.

## Contribution Guidelines for Future Self
1. Avoid committing massive training data files. Use references to public sources (e.g., Kaggle/HuggingFace).
2. Document preprocessing steps clearly.
